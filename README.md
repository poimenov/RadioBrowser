## Description

Photino Blazor application for playing radio stations from [radio browser](https://www.radio-browser.info/)

Works on Linux and Windows

### Prerequisites

.Net 8.0 SDK 

### Installation

Clone the repo:

```bash
git clone https://github.com/poimenov/RadioBrowser.git
```
